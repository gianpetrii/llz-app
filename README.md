# LLZ macro y finanzas

## Descripción del proyecto

Sitio **landing** para **LLZ**, consultora de macroeconomía, finanzas y política monetaria: mensaje institucional, multi-idioma (ES/EN), tema claro/oscuro y presentación visual cuidada.

## Problema que resuelve

Ofrece presencia web profesional y clara para un servicio de asesoría que necesita transmitir credibilidad y alcance internacional sin depender de plantillas genéricas desconectadas de la marca.

## Stack

- Next.js 14, React, TypeScript  
- Tailwind CSS, shadcn/ui, next-intl, next-themes  

## Requisitos

- Node.js LTS  

## Instalación

```bash
npm install
npm run dev
```

El servidor de desarrollo y producción usan el puerto **3005** ([http://localhost:3005](http://localhost:3005)).

Scripts: `build`, `start`, `lint`, `format`.

## Variables de entorno

No suelen hacer falta secretos para la landing; si agregás integraciones, usá `.env.local` según lo que lea el código.
