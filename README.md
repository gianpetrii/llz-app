# LLZ macro y finanzas

Landing page para LLZ - Consultora especializada en macro, finanzas y política monetaria.

## Características

- Soporte multiidioma (Español/English)
- Tema claro/oscuro
- Animaciones suaves y profesionales
- Diseño responsive
- Cards flotantes animadas en hero

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- next-intl

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3005](http://localhost:3005)

## Producción

```bash
npm run build
npm start
```
